Bienvenue dans l'application de gestion de la POBOT Junior Cup.

Elle permet de :

  • saisir en temps réel les scores des différentes épreuves
  • suivre le déroulement de la compétition par rapport au planning

  • serveur développé en Python sur la base du framework Tornado
  • programmation client :
    • jQuery
    • plugin jQuery validation
  • built with Bootstrap and Bootswatch Slate
